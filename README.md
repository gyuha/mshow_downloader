# MShow downloader



## Description
Download the comics registered on the web site



## Use
### Install files...

#### Windows
```cmd
$ python -m venv venv
$ .\venv\Scripts\activate.bat
$ pip install -r requirements.txt
```

#### Linux
```bash
$ sudo apt-get install chromium-chromedriver
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```

------

## How to use
```cmd
$ python main.py
```
